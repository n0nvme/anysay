# Ricksay
like ponysay but with Ricks

## Installation

```bash
git clone https://github.com/n0nvme/ricksay.git
cd ricksay
pyenv local 3.7.4
poetry install
poetry run python src
```



## Dependencies
- [poetry](https://github.com/python-poetry/poetry)
- [pyenv](https://github.com/pyenv/pyenv)
- python ^3.7
